异常和错误

  1. Exception 异常——人为规定的一种不正常现象

    • 编译时异常
    • 运行时异常

  2. Error 错误 —— 计算机根本处理不了了，内存溢出

常见运行时异常

1. InputMisMatchException

  • 输入类型不匹配

2. ArrayIndexOutOfBoudsException

  • 数组越界

    1. 静态初始化

    2. 动态初始化

3. NegativeArraySizeException

  • 数组长度不合法：负数

4. NullPointException

  • 空指针异常 / 空指向异常

    引用为null可以，还拿来使用就不行了

5. NumberFormatException

  • 转化问题：如输入String:abc无法转化成Int:数字

6. StackOverflowError

  • 栈溢出错误 —— 方法间来回调用

  • 堆内存溢出错误 OutMemoryError

7. ArithmeticException

  • 数字异常 整数/0
    • 创建对象时未设置参数，方法内有除以参数的计算，而参数未赋值时默认为0

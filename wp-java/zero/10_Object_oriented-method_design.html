类中的方法

类中的方法 —— 做一件事情

结构

  • 必要结构：
  • 根据返回值和参数内容有无分类

  1. 无参数无返回值

  2. 无参数有返回值

  3. 有参数无返回值

  4. 有参数有返回值

属性和方法

人类有名字 有年龄 有性别 —— 属性

人类能吃饭 能说话 能学习 —— 方法

  • 方法 —— 做了一件事
  • 返回值 —— 这件事做完了 留下一个结果
  • 参数 —— 做事情前必须提供的条件

    2. 通过对象调用方法
